## 概要
このゲームは
- レポートゲーム(仮)
- しりとりゲーム(仮)
の2つを遊べるゲームです。
このファイル(readme.txt)と同じ階層にあるstart.pyとmaterials(ディレクトリ)を同一階層に配置した上でCUI上から
	$ python start.py
等のコマンドでstart.pyを実行してください。
未完成なので実行は自己責任でお願いします(よっぽど無茶な動かし方をしなければ問題ないと思います)。
バグ/不自然な挙動を見つけた場合は報告をお願いします。

-----

## 必須環境
- python3.9.0(最低でもpython3系列を使用してください。特にpython2系では動作しない可能性が高いです)
- pygame2.0.0(2.0.0で新規追加された機能を使用しているので過去verでは動作しない可能性が高いです)

## ゲーム解説
### レポートゲーム(仮)
穴抜きされた単語とその単語の説明が表示されるので、その単語を表す英語を解答するゲーム。
時間内に単語を入力しきればレポートが完成(ゲームクリア)、時間内に単語を入力しきれなかったor一定数以上単語を間違えるとレポート提出失敗(ゲームオーバー)となる。
※実装できていない仕様等
- ”単語の説明”の改良(データセットの改良)
単語の説明に穴抜きする前の単語が含まれてしまっているのでそれを修正
- 時間管理(タイマー)の追加(時間制限)
画面右上に表示される数字は現状ダミー
- リザルト設定
結果に対してリザルトの設定を行う(例えば、S,A,B,C,Dの5段階評価等)
### しりとりゲーム(仮)
名詞, 動詞, 形容詞をしりとり形式で入力していくゲーム
時間内に正解(入力)すべき単語数が決まっており、所定の単語数に到達するまでに間違えた回数を記録している。
※実装できていない仕様等
- リザルト設定
結果に対してリザルトの設定を行う(例えば、S,A,B,C,Dの5段階評価等)
