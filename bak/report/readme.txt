# 備考
せっかく単語を穴抜きしたのに説明の方に単語が普通に見えてるのはご愛嬌
記述に統一性があれば取り除くことが可能だけどまだそこまで辿りついてない(未確認)

現状、入力された文字列(解答)と単語(アルファベット)が完全一致の場合正解(True)を返すような仕様になっている
なので、大/小文字と半角スペースも厳密に書かないと正解判定にならない
正規表現使えばその辺の問題は解決できそうだけど使いたくないです(トラウマ)
