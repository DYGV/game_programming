# ファイル
- user_interface.py
pygameでウィンドウを生成して画面中央にテキストを表示するプログラム
PC上の任意のディレクトリに設置してコマンドプロンプト等からpythonコマンドで実行する
(インタラクティブシェルに読み込まなくても動作可能)

- align.py
pygameのSurfaceオブジェクトを画面に描画する際の位置を左, 右, 中央(center), 上, 下, 中央(middle)で指定可能にするAlignクラスが記述されているプログラム

- colors.py
ウェブカラー16種のRGB値をタプル型の定数として記述してあるプログラム。
またRGB値のカラー(タプル型)の補色を求めて返す関数を含む。
